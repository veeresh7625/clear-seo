# Clear SEO

Clear SEO is a fast, clean, and beginner-friendly SEO audit tool that helps analyze a website’s basic search engine optimization health and highlights important improvements.

## Features

* Website SEO analysis
* Basic on-page SEO checks
* Meta tag inspection
* Title and description review
* Heading structure analysis
* Image alt text checks
* Performance-related SEO insights
* Simple and easy-to-read report output

## Tech Stack

* Frontend: React / Next.js
* Backend: FastAPI / Python
* Styling: Tailwind CSS
* Data Handling: REST APIs
* Deployment: Vercel / Render / Railway

## Installation

### 1. Clone the repository

```bash
git clone https://github.com/veeresh7625/clear-seo.git
cd clear-seo
```

### 2. Setup backend

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

### 3. Setup frontend

```bash
cd ../frontend
npm install
npm run dev
```

## Usage

1. Open the frontend in your browser.
2. Enter a website URL.
3. Click on the analyze button.
4. View the SEO report and suggestions.
